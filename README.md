# FromScratch
This repo will have from scratch implementations of famous papers, tools, and websites in the AI ecosystem. From scratch means that the code does not use specialized libraries or specific API. Every line of code should be completely accessible to anyone with a basic knowledge of libraries such as numpy and pytorch or basic web deployment tools like node js.

In general, the idea is that every implementation will be written directly from the information in the associated paper and documentation as needed.
